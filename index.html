FLRW Metric

$ds^2 = a^2(t) (- d\tau^2 + d\Sigma^2)$

with scale factor $a(t)$

Hubble parameter $H = \frac{\dot a}{a}$

comoving Hubble radius $(a H)^{-1}$

comoving horizon: $\tau = \int_0^t \frac{dt'}{a(t')} = \int_0^a (a' H)^{-1} d \ln a'$

maximum distance light can travel in time $t$

separation by ...

... comoving Hubble radius $(a H)^{-1}$: can't talk now

... comoving horizon $\tau$: could never have talked

both increase with time!*

* Well, not really, but we'll get to that

Horizon Problem

If $(a H)^{-1}$ grows in time...

... stuff entering the Hubble volume today
cannot have been in contact before!

... but the CMB is in thermal equilibrium!
